# Tsundere

## Overview

[Yahoo Hack Day 2017](http://hackday.jp) というハッカソンで開発した積読管理アプリ

## Detail

買っただけで読まずに放っておいてしまう本--積読本--を管理するためのiOSアプリケーションです。
Facebookアカウントと連携することで友達の積読本や読了本を見ることができ、その結果をランキングとして表示します。
